## YayaPL - Yet another "Yet another" Programming Language

本项目与更多仍在计划中（存疑）的项目同属 Yaya 系列。

Yaya 由两个“YA”组合而成，后者是很常见的前缀，是“Yet Another”的缩写，典型案例包括 YACC（Yet Another Compiler Compiler）和 YARV（Yet Another Ruby VM）等。考虑到 YA 前缀太常见，又懒得想别的名，最终决定连用两个 YA，这样就能极大概率避开重名。
